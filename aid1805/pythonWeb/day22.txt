1.cookies 和 session
  1.什么是cookies?
    cookies是一种数据存储技术，将一段文本保存在客户端(浏览器)的一种技术，并可以长期的保存。
  2.cookies的使用场合
    1.保存登录信息
    2.保存用户的搜索关键词
  3.django中使用cookies
    1.设置cookies(保存数据到客户端)
      1.不使用模板
        resp = HttpResponse('给客户端的响应')
        resp.set_cookie(key, value, expires)
        return resp
            key:cookie的名称
            value:保存的cookie的值
            expires:保存的时间，以秒s为单位
        ex:resp.set_cookie('uname', 'zsf', 60*60*24*1)
      2.使用模板
        resp = render(request, 'xx.html', locals())
        resp.set_cookie(key, value, expires)
        return resp
      3.重定向
        resp = HttpResponseRedirect('/login/')
        resp.set_cookie(key, value, expires)
        return resp
    2.获取cookies(从客户端获取数据)
      request.COOKIES
2.session - 会话
  1.什么是session?
    session(会话)，实际上是服务器为每个浏览器临时开辟的一段空间，用于保存相关的请求信息。
  2.session的应用场合
    session也是为了存放数据而存在的，通常会把服务器端要用到的数据保存进session里。
3.django中使用session(在浏览器cookie中以sessionid的形式体现)
  1.设置session的值
    request.session['key'] = 值
    request.session.set_expiry(time):设置过期时间，如果设置为0则表示浏览器关闭即失效。
  2.获取session的值
    request.session.get('key')或者request.session['key']
  3.删除session的值
    del request.session['key']
  4.settings.py中 有关session的设置
    1.SESSION_COOKIE_AGE = 60 * 30
    2.SESSION_EXPIRE_AT_BROWSER_CLOSE = True
